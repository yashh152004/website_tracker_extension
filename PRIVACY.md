# Privacy Policy – Website Time Tracker

**Website Time Tracker** does not collect, transmit, or share any personal information.

- All browsing time data is stored **locally** in the user’s browser.  
- Data is used **only** to display daily website usage statistics.  
- **No data is sent** to any external servers or third parties.  
- Permissions (Tabs / Storage) are used solely for the functionality of tracking website usage.
